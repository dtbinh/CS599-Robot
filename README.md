# CS599-Robot-P4

### How to compile?
```
make all
```

### How to start the Player/Stage?
```
player player/multi.cfg
```

### How to run the robots?
```
sh run_robot.sh
```

### How to run the task manager?
```
sh run_taskmanager.sh
```
